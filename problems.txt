5、JavaScript中如何对一个对象进行深度clone
  function cloneObject(o) {
      if(!o || 'object' !== typeof o) {
          return o;
      }
      var c = 'function' === typeof o.pop ? [] : {};
      var p, v;
      for(p in o) {
          if(o.hasOwnProperty(p)) {
              v = o[p];
              if(v && 'object' === typeof v) {
                  c[p] = Ext.ux.clone(v);
              }
              else {
                  c[p] = v;
              }
          }
      }
      return c;
  };

30、jQuery中 detach() 和 remove() 方法的区别是什么?
   尽管 detach() 和 remove() 方法都被用来移除一个DOM元素, 两者之间的主要不同在于
   detach() 会保持对过去被解除元素的跟踪,因此它可以被取消解除,
   而 remove() 方法则会保持过去被移除对象的引用.
